# TechVerse

Latest technology for everyone
## Project Structure

```
affiliate-blog/
├── index.html
├── review.html
├── styles.css
├── script.js
└── README.md
```

## Features

- Header with logo and navigation menu
- Hero section introducing the blog
- Featured article section with iPad 11th Gen and Samsung Galaxy Tab S9 FE comparisons
- Blog post cards with realistic gadget-focused descriptions
- Dedicated product review template page with:
  - Product overview
  - Pros and cons
  - Specifications table
  - Who should buy guidance
  - Affiliate buy button placeholders
  - Product comparison section
- Affiliate CTA buttons and placeholder affiliate URLs
- Sidebar categories for gadget topics
- Footer with social media links
- Mobile responsive layout with collapsible menu

## Run Locally

Open `index.html` in your browser for the homepage and `review.html` for the product review template.
